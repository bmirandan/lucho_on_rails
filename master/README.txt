Prototipo DUC
Lifeware

Contiene dos carpetas
- DUC Client
- DUC Server

Corresponden a una aplicacion web desarrollada en AngularJs (1.5.0) y a una API desarrollada en NodeJs respectivamente.

Requisitos Previos:
-NodeJs. Probado en versión 4.4.7

Instalación:
- DUC Client: Requiere un servidor http. El servidor debe ejecutarse en carpeta "DUC Client/source". Se recomienda usar http-server de NodeJs
- DUC Server: En carpeta "DUC Server/source", abrir consola y ejecutar comando: "node main.js" (sin comillas)

USO:
- DUC Client: Abrir URL dada por el servidor http
- DUC Client: Abrir <tu ip>:3000, deberia aparecer: "There is nothing to see here!"

Recomendación:
Uso de http-server: para ello debe tenerse instalado nodejs, en la consola ejecutar comando: "npm install -g http-server", luego, dirigirse a carpeta donde se desea montar el servidor y escribir comando http-server.
